# HMER using Graph Neural Networks

## 📌 Project Overview

Handwritten Mathematical Expression Recognition (HMER) is a challenging computer vision task due to the two-dimensional structure and spatial relationships between mathematical symbols.

This project develops a hybrid deep learning framework combining DenseNet, Graph Neural Networks (GNN), and Tree-Aware Mathematical Expression Recognition (TAMER) to accurately recognize handwritten mathematical expressions.

---

## 🎯 Objectives

- Recognize handwritten mathematical expressions
- Improve symbol recognition accuracy
- Capture spatial relationships using Graph Neural Networks
- Preserve mathematical structure using TAMER
- Evaluate performance on the CROHME dataset

---

## 📂 Dataset

**CROHME (Competition on Recognition of Handwritten Mathematical Expressions)**

Contains handwritten mathematical expressions with symbol-level annotations.
